# Data directory

This directory contains machine-readable structures and figure-related materials from the manuscript workflow.

- `structures/initialization/`: clean slab and representative automatically initialized structures.
- `structures/selected_sets/`: selected or checking structures used during validation/selection tests.
- `structures/validation_sets/`: validation structures in `extxyz` format.
- `figure_data/`: generated figure outputs and location for CSV files underlying manuscript figures.

For the final public repository, place the CSV/source tables underlying each manuscript figure in `figure_data/` and update the file manifest accordingly.
