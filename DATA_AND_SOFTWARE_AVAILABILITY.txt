Data and Software Availability

The data and software materials necessary to reproduce and evaluate the main results of this study are available at the public GitHub repository: [GitHub repository link]. The repository includes machine-readable structural datasets, validation structures, selected active-learning structures, numerical data and figure-generation materials where applicable, representative DFT input templates excluding license-restricted VASP pseudopotential files, NequIP training configuration templates and model metadata, and scripts for post-processing, validation, coverage-dependent adsorption-energy analysis, reaction-path analysis, and figure generation. The repository also includes reference implementations of selected non-sensitive components, including Delaunay/grid-based dual-channel initialization building blocks and SOAP/KPCA/FPS diversity-driven selection.

The coupled MH-MC sampling code developed in this work is an in-house research implementation that is under active development for future software release and is not fully publicly distributed at this stage. To support reproducibility, the manuscript and the GitHub repository provide the algorithmic description, pseudocode-level workflow, hyperparameters, input/output formats, representative examples, and sampled structures used for NequIP model training and validation. Access to the in-house implementation for academic evaluation may be arranged through the corresponding authors subject to institutional and licensing restrictions. The production calculations used VASP and NequIP. VASP pseudopotential files are not redistributed because of licensing restrictions, but all relevant calculation settings and element orders are provided.
