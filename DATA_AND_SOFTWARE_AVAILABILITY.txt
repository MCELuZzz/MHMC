Data and Software Availability

The data underlying this study are available at the public GitHub repository: https://github.com/MCELuZzz/MHMC. The scripts and code used to generate, process, analyze, and visualize the results are available at the same repository. The repository includes representative structural datasets, selected active-learning structures, validation data, DFT input templates excluding license-restricted VASP pseudopotential files, NequIP configuration files and model metadata, analysis scripts, figure-generation files, and reference workflow materials for the coupled MH-MC active-learning procedure.
